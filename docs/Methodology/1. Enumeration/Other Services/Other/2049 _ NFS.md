After seeing NFS port is open we start with seeing what's on there
showmount -e 10.10.253.194

In this case /users was open through NFS.

Make a new directory for the drive we'll be taking locally for now:
mkdir /mnt/new_back

Mount it: 
mount -t nfs 10.10.253.194:/users /mnt/new_back -o nolock

Then do what you have to do with it.
When you're done, make sure to unmount it. If not, it can cause some annoying issues:
umount /mnt/new_back
