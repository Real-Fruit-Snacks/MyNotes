If you have file upload, but php files are disallowed, then try this technique.
It can make apache interpret arbitrary files as php codes with the usage of uploading a .access file to potentially change configuration settings on server side. 

echo ‘AddType application/x-httpd-php.cth’ > .htaccess

then upload file to server
then grab a php webshell and call it anyname.cth and this could potentially allow your php code now to execute!
