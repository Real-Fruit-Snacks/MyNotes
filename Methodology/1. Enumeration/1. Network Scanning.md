## Initial Network Scanning
### Individual Scans
```bash
cd /root/Labs/<RoomName>/<IPAddress>/network_scans

nmap -p- -Pn <TargetIP> -v --min-rate 1000 --max-rtt-timeout 1000ms --max-retries 5 -oN nmap_Ports.txt && sleep 5 && nmap -Pn $target -sV -sC -v -oN nmap_Services.txt && sleep 5 && nmap -T5 -Pn $target -v --script vuln -oN nmap_Vuln.txt 
```
### Full Scan
```bash
cd /root/Labs/<RoomName>/<IPAddress>/network_scans

rustscan --ulimit 5000 -a $target -- -sC -sV -Pn -oN nmap_Full.txt
```
